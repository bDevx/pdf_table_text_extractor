1. For Windows:
You can download a pre-built Poppler for Windows from the following GitHub repository:
https://github.com/oschwartz10612/poppler-windows/releases/

Download link for Poppler (Windows binaries): Poppler for Windows
Steps:
Visit the link above and download the latest release ZIP file under Assets (look for something like poppler-xx.x.x.zip).
Extract the contents of the ZIP file.
After extracting, navigate to the poppler-xx.x.x\bin folder.
Copy the path to the bin folder (e.g., C:\path\to\poppler-xx.x.x\bin).
